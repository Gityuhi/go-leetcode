# go-leetcode


## 進め方
1. 各項目(Arraysなど)に取り掛かったら、該当する項目のyoutubeや記事やudemyを軽く見て、概念を理解する。
2. 問題を見て、自力で考える。
2.5解説動画見る前に、日本語でどこまで実装できるかを考える
3. easyは15分、medium/hardは40分くらい考えても分からなければ、解説動画を見る。
4. 再度自力で解いていく。
5. 最終的にACと計算量の観点での言語化、使用したデータ構造とアルゴリズムの選択理由の言語化ができたら、終了
